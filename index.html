⭐ Flagship Project RAG Agentic AI MCP Server
CodeIntel-RAG — Agentic Codebase Intelligence

Self-hosted agentic RAG platform for natural-language querying over Python, Java, JavaScript, TypeScript & C++ codebases. Custom 5-signal hybrid retrieval combining BGE + FAISS embeddings, BM25, call-graph traversal, file-metadata filtering, and name-token concept matching. LangGraph ReAct agent + spec-compliant FastMCP server with 5 tools including a Bandit + regex security scanner with CWE/OWASP Top 10 mapping. Compatible with Cursor and Continue.dev over SSE. FastAPI backend with SSE streaming, query caching, Docker Compose, Whisper voice input, and interactive Pyvis call graphs.

Hit@5: 1.00 · MRR: 0.845 · +8.7% vs semantic baseline

Disk-backed query cache with automatic invalidation and structured JSON-lines execution logging. 92-test suite enforced via GitHub Actions CI.
